# Pokémon Battle Game Sounds

This directory contains sound effects for the battle system.

## Sound Files Needed:
- hit.mp3 - Normal attack hit
- critical.mp3 - Critical hit sound
- ko.mp3 - Knock out sound
- super-move.mp3 - Super move sound
- heal.mp3 - Healing sound
- switch.mp3 - Pokémon switch sound
- victory.mp3 - Victory fanfare
- select.mp3 - Menu selection sound
- super-effective.mp3 - Super effective hit
- not-very-effective.mp3 - Weak hit sound

## Temporary:
For now, the SoundManager will create simple synthesized sounds using Web Audio API as fallback until proper sound files are added.
